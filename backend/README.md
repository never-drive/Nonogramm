# Nonogram Backend

## Setup

- `pip install -r ./requirements.txt`
- `flask run`

## Run samples

- `python nono_small.py`
- `python nono_mouse.py`
- `python nono.py`

## Run some backend URLs

- `http://127.0.0.1:5000/`
- `http://127.0.0.1:5000/about`
- `http://127.0.0.1:5000/capitalize/hello`

## Testing

- `python -m pytest`
